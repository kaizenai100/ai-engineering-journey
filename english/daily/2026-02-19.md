## New collocations 
customer support = 客服/客户支持
technical support = 技术支持
production issue = 线上问题（生产环境问题）

## Key word
issue = 问题/工单
affect = 影响

## Sendtences
We have an issue in production. = 我线上出问题了。
I opened an issue o track the bug. = 我打开了一个工单来跟踪这个bug
This issue affect customer. = 这个问题影响客户
This issue affect performance. = 这个问题影响性能
This issue affect users. = 这个问题影响用户
